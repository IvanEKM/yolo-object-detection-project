# Система распознавания объектов в городской среде

## Описание проекта

Проект представляет собой прототип системы компьютерного зрения для мониторинга городской среды.  
Используется предобученная модель YOLO для автоматического распознавания объектов на изображениях.

Система может применяться для анализа инфраструктуры, транспорта и потоков людей в рамках задач умного города и экологического мониторинга.

## Используемые технологии

- Python
- Google Colab
- Ultralytics YOLOv8
- Computer Vision

## Как запустить

1. Открыть файл yolo_project.ipynb в Google Colab
2. Запустить ячейку с кодом
3. Модель автоматически скачает веса
4. Выполнится распознавание объектов
5. Результат сохранится в папке:

runs/detect/predict/

## Результат

Модель успешно распознаёт объекты на городской сцене: людей и транспорт.

Это демонстрирует применимость предобученных моделей для задач визуального мониторинга.

## Вывод

Предобученные модели позволяют быстро создавать рабочие прототипы систем анализа окружающей среды без длительного обучения нейросетей.

Проект демонстрирует базовый подход к построению IT-решений для мониторинга экосистемы.
